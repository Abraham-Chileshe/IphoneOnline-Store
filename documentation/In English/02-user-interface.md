# User Interface & Platform Usage Guide

This document explains how users interact with the iPhone Online Store and provides a step-by-step example of the purchasing process.

## 1. Product Discovery
- **Search:** Find products instantly using the global search bar.
- **Filtering:** Use the sidebar to filter by **City** and **Category**.
- **Interactive Toggles:** Switch between **Light/Dark mode** and **English/Russian** at any time.

## 2. Step-by-Step Example: Placing an Order

To help you understand the customer journey, here is how a user places an order:

1. **Browsing:** The user selects a product (e.g., iPhone 15 Pro).
2. **Contacting (Optional):** If they are a guest and have questions, they click **"Contact"** to see the store owner's WhatsApp and Email in a popup.
3. **Cart:** The user clicks **"Add to Cart"**. They can see their items by clicking the cart icon in the header or sidebar.
4. **Checkout:** In the "My Cart" section, the user clicks **"Place Order"**.
   - *Note:* If the user is not logged in, they will be redirected to the **Login/Register** page.
5. **Confirmation:** Once logged in and the order is placed, the user sees a success message.
6. **Tracking:** The user goes to **Profile > Orders** to see their new order marked as **"Pending"**.

## 3. Post-Order Features
- **Order Details:** Clicking on an order shows the full summary.
- **Direct Assistance:** Every order card includes a **"Need help?"** section. Users can click the admin's phone number to start a call/WhatsApp or click the email to send a message regarding that specific order.
- **Wishlist:** Users can click the heart icon on any product to save it for later viewing in their profile.
