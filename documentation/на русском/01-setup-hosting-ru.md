# Руководство по настройке и хостингу

Это руководство содержит технические подробности о среде и шагах, необходимых для развертывания и настройки онлайн-магазина iPhone.

## Технологический стек
- **Фреймворк:** Laravel 11
- **Интерфейс в реальном времени:** Laravel Livewire 3
- **База данных:** MySQL / MariaDB
- **Язык программирования:** PHP 8.2+
- **Фронтенд:** Ванильный CSS (современная дизайн-система), Alpine.js (для интерактивности)

## Требования к развертыванию
1. **PHP:** Версия 8.2 или выше с необходимыми расширениями (BCMath, Ctype, Fileinfo, JSON, Mbstring, OpenSSL, PDO, Tokenizer, XML).
2. **База данных:** MySQL 5.7+ или MariaDB 10.3+.
3. **Веб-сервер:** Apache или Nginx с включенной перезаписью URL (mod_rewrite).
4. **Composer:** Необходим для управления зависимостями.

## Начальные шаги по настройке

### 1. Конфигурация окружения
Скопируйте `.env.example` в `.env` и настройте учетные данные вашей базы данных:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=имя_вашей_базы_данных
DB_USERNAME=ваше_имя_пользователя
DB_PASSWORD=ваш_пароль
```

### 2. Установка зависимостей
Выполните следующую команду для установки зависимостей Laravel:
```bash
composer install
```

### 3. Генерация ключа приложения
```bash
php artisan key:generate
```

### 4. Миграции и заполнение базы данных (Seeding)
Чтобы создать структуру базы данных и заполнить ее начальными данными (товары, города и администратор), выполните:
```bash
php artisan migrate --seed
```

## Доступ к панели администратора
После выполнения сидинга (seeding) базы данных вы сможете войти в панель управления, используя следующие данные:
- **URL:** `ваш-домен.com/login`
- **Email:** `Romanvov@site.com`
- **Пароль:** `Romanvov@Rinac`

## Обслуживание
- **Ссылка на хранилище:** Убедитесь, что папка storage связана с public для отображения загруженных файлов:
  ```bash
  php artisan storage:link
  ```
- **Файлы локализации:** Все переводы находятся в директории `/lang` (`en.json`, `ru.json`).
